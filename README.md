# Customer-Behavior-Analysis-Dashboard
1. Project Title-
Customer Behavior Analysis Dashboard | Power BI Business Intelligence Project
2. Short Description : 
The Customer Behavior Analysis Dashboard is an interactive Power BI project designed to analyze customer purchasing behavior, spending patterns, product performance, and demographic trends. By transforming raw transactional data into actionable insights, the dashboard enables businesses to make informed decisions in marketing, sales, and customer engagement.
3. Tech Stack : 
Power BI :- Interactive Dashboard & Data Visualization
Power Query :- Data Cleaning & Transformation
DAX (Data Analysis Expressions) :- KPI Calculations & Business Metrics
Microsoft Excel / CSV :- Data Source 
Data Modeling :- Relationship Management & Data Analysis 
4. Data Source :- Kaggle 
Dataset includes:
Customer ID
Product Category
Purchase Amount
Customer Age
Gender
Shipping Type
Payment Method
Customer Rating
Discount Applied
Purchase Season
Customer Location
5. Features / Highlights :
   Business Problem:
Retail businesses often collect large volumes of customer transaction data, but identifying meaningful purchasing patterns and customer preferences from raw data is difficult. Manual analysis is time-consuming and limits strategic decision-making.
